# Assets Folder - Image Requirements

Place the following images in this folder to complete your app setup:

## Required Images:

### 1. **icon.png** (App Icon)
- **Size**: 1024x1024 pixels
- **Format**: PNG with transparency
- **Purpose**: Main app icon for both iOS and Android
- **Design**: Should be your SubTrack AI logo

### 2. **adaptive-icon.png** (Android Adaptive Icon)
- **Size**: 1024x1024 pixels
- **Format**: PNG with transparency
- **Purpose**: Foreground layer for Android adaptive icon
- **Design**: Should be your logo centered, keep important elements in the center 66% of the canvas

### 3. **splash.png** (Splash Screen)
- **Size**: 1284x2778 pixels (or at least 2048x2048)
- **Format**: PNG
- **Purpose**: Loading screen shown when app starts
- **Design**: Your logo/branding with background color #0B1220

## Quick Tip:
You can use online tools like:
- https://www.appicon.co/ - Generate all icon sizes
- https://icon.kitchen/ - Icon generator
- Figma or Canva - Design your icons
